Proyectos

Cada módulo puede contratarse en forma separada o integrado como plan.

Proyecto 1: Securización y Reinstalación Servidor Tango/SQL

Instalación limpia de Windows Server, SQL actualizado, VPN, perfiles SQL y RDP de contingencia.

Proyecto 2: Migración File Server → SharePoint

Relevamiento, migración por fases, normalización de permisos y adopción de OneDrive/Teams.

Proyecto 3: AD y Migración de Perfiles

Integración de PCs al dominio, migración de perfiles, restricción de privilegios y cliente Tango.

Proyecto 4: Antivirus ESET en Estaciones

Despliegue del agente ESET, políticas centralizadas, escaneo de correo y webfilter.

Proyecto 5: Licenciamiento & Control de Aplicaciones

Inventario de aplicaciones, licencias Windows/Office/Server y políticas de instalación.

Proyecto 6: Producto Comercial y Página Web

Empaquetado de servicios como abono mensual y material comercial para la web.
